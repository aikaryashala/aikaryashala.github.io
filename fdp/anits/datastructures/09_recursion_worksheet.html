    result = 0
    while n > 0:
        result = result + n % 10
        n = n // 10
    return result

A function that calls itself. Pay close attention to when work happens — before the recursive call, or after it.

01 What does this program print? 
def fun(n):
    if n == 0:
        return
    print(n, end=" ")
    fun(n - 1)

fun(4)
Output
