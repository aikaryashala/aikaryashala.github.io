① The Wire is Shared

On a hub, Wi-Fi, or via ARP-spoofing on a switch, every NIC on the segment physically receives every frame. Normally, NICs drop frames not addressed to them. In promiscuous mode, they keep them all.

② Layers, Layers, Layers

Each packet is a stack of headers: Ethernet → IP → TCP → Application. A sniffer reads them top-to-bottom. Click a captured frame and expand each layer to see what's in the headers.

③ Encryption is the Defense

HTTP, FTP, Telnet send passwords in plaintext — Eve reads them instantly. HTTPS, SSH, SFTP wrap the payload in TLS/SSH; Eve still sees the headers, but the data is cipher-gibberish.
